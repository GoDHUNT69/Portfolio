## What This Is

An interactive, single-page personal portfolio тАФ built to feel like a professional vCard rather than a static resume site. Clean dark surfaces, section-based navigation, filtered project showcase, and a live AI chat widget that's ready to connect to any LLM backend.

> Built with Next.js 16, React 19, Framer Motion, and custom CSS. Deployable to Vercel in one click.

## Getting Started

### Prerequisites

- Node.js 20.x
- npm

### Install & Run

```bash
# Clone the repo
git clone https://github.com/your-username/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Build for Production

```bash
npm run build
npm run start
```

## Customization

Most content lives in `app/page.js` as structured arrays тАФ no CMS, no database, just code you can edit directly.

| File | What to Change |
|---|---|
| `app/page.js` | Projects, skills, education, experience, highlights |
| `app/layout.js` | Page title, metadata, SEO |
| `styles/vcard.css` | Layout, card sizing, typography scale, responsive behavior |
| `styles/globals.css` | Shared theme variables, base colors |

### Common Extensions

- Add real project links and live demo URLs to the projects array
- Replace the placeholder contact form with [Resend](https://resend.com), [Formspree](https://formspree.io), or similar
- Add a downloadable resume via a public PDF link
- Integrate Plausible or Vercel Analytics for visitor tracking
- Pull content from a headless CMS (e.g. Sanity, Notion API) instead of hardcoding

## AI Chat Widget

The floating chat widget is wired up and ready тАФ it just needs a real model behind it.

**Current behavior:**

```
User message  тЖТ  POST /api/chat  тЖТ  handleChat()  тЖТ  "AI module under construction."
```

**To connect a real LLM**, edit these two files:

- `app/api/chat/route.js` тАФ the Next.js API route
- `lib/chatService.js` тАФ the response logic (currently returns a placeholder)

**Recommended upgrades:**

```
Connect OpenAI / Anthropic / Gemini            тЖР swap in lib/chatService.js
Ground responses on portfolio context          тЖР pass your bio + projects as system prompt
Stream tokens for real-time feel               тЖР use ReadableStream in route.js
Persist conversation history                   тЖР localStorage or a simple DB
Rate-limit abusive requests                    тЖР upstash/ratelimit or Vercel Edge Config
```

## Styling System

Two CSS files power the visual design:

**`styles/vcard.css`** тАФ the full layout system
- Sticky sidebar + main content composition
- Section card styling
- Project filter UI
- Chat widget
- Mobile responsive overrides

Design direction: dark layered card surfaces, accent gradients, compact sidebar, rounded panels.

## Deployment

This project is Vercel-native. No special configuration needed.

### Deploy in 3 Steps

1. Push this repo to GitHub
2. Import it at [vercel.com/new](https://vercel.com/new)
3. Keep the default Next.js settings тЖТ **Deploy**

That's it. Environment variables are only needed if you wire up a real LLM API key.
